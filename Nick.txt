SnailTech Socratic Questions

1) How much are you willing to spend on this project?
  $5,000,000

2) Is upgrading the wireless technology an option?
   Can we work around the slower, pre-existing systems?
   -> What in the current system works well that we want to keep?

3) What information does the database contain exactly?
   How do we parse the information?
   -> How much memory does the database have to add new columns/rows?
   -> Would we have to expand the database/server if we had more students?

4) How exactly are you recording attendance right now?
  On paper.

5) Do you have any preferences on how you want us to record attendance?
   Should include at least name and date?
   -> Do we expect preferences to change in the future?
